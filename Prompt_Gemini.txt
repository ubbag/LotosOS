# Complete UI/UX Design System for Lotos SPA Management Platform

## Executive Summary

Design a comprehensive, professional management system for "Lotos SPA" - a luxury massage salon. The system must be intuitive, elegant, and efficient, serving receptionists, therapists, and managers in daily operations. The interface should evoke calmness and professionalism while maximizing operational efficiency through modern, polished UI patterns.

## System Architecture Overview

### Application Type
- **Platform**: Web-based SPA (Single Page Application)
- **Tech Stack**: React + TypeScript + Tailwind CSS
- **Backend**: RESTful API (Fastify + Prisma + PostgreSQL/SQLite)
- **Authentication**: JWT-based session management
- **Target Users**: Receptionists (primary), Managers, Therapists

### Core Modules (13 Main Sections)

1. **Dashboard** - Overview, statistics, and quick actions
2. **Clients (Klienci)** - Client database with visit history
3. **Reservations (Rezerwacje)** - Booking management
4. **Schedule (Harmonogram)** - Therapist availability calendar
5. **Therapists (Pracownicy)** - Staff management
6. **Treatment Rooms (Gabinety)** - Room availability
7. **Services (Usługi)** - Service catalog and pricing
8. **Categories (Kategorie)** - Service categorization
9. **Packages (Pakiety)** - Hour-based service packages
10. **Vouchers (Vouchery)** - Gift voucher management
11. **Reports (Raporty)** - Revenue and analytics
12. **SMS** - Communication module
13. **Settings (Ustawienia)** - System configuration

## Responsive Behavior

### Mobile (<768px)

```
- Sidebar: Drawer overlay with backdrop
- Grid: 1 column
- Statistics: 1 column
- Header: Stacked layout
- Padding: p-4 instead of p-8
- Mobile header visible with hamburger menu
```

### Tablet (768px - 1024px)

```
- Sidebar: May collapse to icons only OR full width
- Grid: 2 columns for cards
- Statistics: 2 columns
- Header: Horizontal flex maintained
```

### Desktop (>1024px)

```
- Sidebar: Always visible, 256px
- Grid: 3 columns for entity cards
- Statistics: 4 columns
- Max content width: 1280px (7xl)
```
